Diario

  • Dia 1
    • Cada uno busca una IA para el proyecto.
    • Puesta en común de cada las herramientas encontradas y selección de la que vamos a usar.
    • Hemos decidido usar Webador. 
  • Dia 2
    • Hoy hemos decidido que uso le vamos a dar a la herramienta, la vamos a usar para dar un ejemplo de página web a los clientes que nos soliciten una, en base a este ejemplo y los cambios sugeridos por el cliente crearemos la página web DEFINITIVA.
  • Dia 3
    • Hemos comenzado el desarrollo del proyecto en el propio Webador, haciendo una página web en la que pondremos todo el contenido de este documento.
    • Se dividirá por secciones para que quede todo más limpio y organizado.
  • Dia 4
    • Hemos empezado a hacer el manual de la web, en nuestro caso Webador, haciendo capturas y guardándolas, para después juntarlas todas con sus indicaciones  y explicaciones.
  • Dia 5
    • Hemos creado la página web que utilizaremos para la presentación del proyecto, copiando y pegando las cosas del documento y adaptándolas a la página web para dejarlo bonito.
