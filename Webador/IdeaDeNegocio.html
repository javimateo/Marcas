Idea de Negocio

Nuestra empresa se encarga de la creación de páginas web para otras empresas o para personas que la necesiten, para ello crearemos una página web con las necesidades del cliente utilizando la inteligencia artificial gracias a Webador, se la mostraremos para que vea una muestra de cómo sería la definitiva, y el cliente nos dirá lo que quiere cambiar o lo que quiere mejorar, cuando sepamos correctamente las necesidades del cliente, nosotros nos encargaremos de crear la página web manualmente.  Una vez hemos creado una plantilla, se la enseñamos al cliente y este nos dirá si nos da el visto bueno o no para empezar a trabajar en esta.  Añadimos el contenido solicitado por el cliente, y dividimos a los trabajadores en grupos de trabajo para que cada grupo se encargue de un apartado en especifico (Periodo de trabajo de 1 semana) Posteriormente hacemos una puesta en común enseñando diseños y bocetos para dar con un diseño común para la web y así entonces ponernos manos a la obra. El proyecto lo dividimos en 3 fases, cuando las vamos completando se lo mostramos al cliente y le explicamos el porque de cada cosa, así podremos ver que cosas le gustan, cuales no, que cambiaria, que añadiría, etc...  Así garantizamos la conformidad del cliente y sea uno mas del equipo. En conclusión, utilizaremos la página web para crear una “Maqueta” de la página web de cada cliente, para luego trabajar sobre esta e ir modificándola.
